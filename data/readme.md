# Data

This folder contains all datasets used in the project, organized by stage.

## Subfolders
- `raw/`  
  Original, unprocessed multilingual data collected from social media platforms.

- `processed/`  
  Final cleaned and balanced dataset used for training and evaluation.

- `debug/`  
  Small subsets of the data used for fast testing and debugging.

- `probs/`  
  Reserved for storing model prediction probabilities or inference outputs.
