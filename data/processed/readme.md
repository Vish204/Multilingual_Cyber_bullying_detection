# Processed Data

This folder contains the final dataset used for model training and evaluation.

## Files
- `train_data.csv`  
  Training split of the dataset.

- `val_data.csv`  
  Validation split used during training.

- `test_data.csv`  
  Test split used for final evaluation.

- `multilingual_dataset.csv`  
  Complete processed dataset before splitting.

- `dataset_stats.json`  
  Statistical summary of the dataset including:
  - class distribution
  - language distribution
  - platform distribution

This dataset is balanced and multilingual.
