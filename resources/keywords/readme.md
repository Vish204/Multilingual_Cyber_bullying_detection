# Keyword Resources

This folder contains keyword-based resources for multilingual cyberbullying detection.

## Files
- `complete_multilingual_database.json`  
  Comprehensive keyword database covering all supported languages.

- `consolidated_keywords.json`  
  Merged keyword list used during preprocessing and analysis.

- `english_keyword_base.json`  
  Base keyword list used as the source for translations.

- `failed_translations.json`  
  Records keywords that failed during translation.

## Subfolder
- `multilingual_keywords/`  
  Language-specific keyword files (one JSON per language).
